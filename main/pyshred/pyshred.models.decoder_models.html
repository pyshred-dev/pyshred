Bases: AbstractDecoder

Flexible Multilayer Perceptron (MLP) implementation.

A fully connected decoder that maps a low-dimensional latent space back to a high-dimensional state with user-defined hidden layers.

pyshred.models.decoder_models.unet_model module#

class pyshred.models.decoder_models.unet_model.UNET(conv1: int = 256, conv2: int = 1024)[source]#

Bases: AbstractDecoder

1D U-Net style convolutional decoder for SHRED.

Uses a series of 1D convolutions to decode latent representations back to the physical state space. Particularly suitable for spatially-structured data.

Parameters:

  • conv1 (int, optional) – Number of channels in first convolutional layer. Defaults to 256.

  • conv2 (int, optional) – Number of channels in second convolutional layer. Defaults to 1024.

Variables:

  • c1 (int) – First convolution layer channel count.

  • c2 (int) – Second convolution layer channel count.
