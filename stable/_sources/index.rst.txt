PySHRED documentation
=====================


.. Version: |version|

**Useful links:** `Installation <https://pyshred-dev.github.io/pyshred/stable/user_guide/installation.html>`_ | 
`Source repository <https://github.com/PyShred-Dev/PyShred>`_ |
`Issue tracker <https://github.com/pyshred-dev/pyshred/issues>`_

PySHRED is an open-source software implementing the SHallow REcurrent Decoder (SHRED) architecture for sensing applications. PySHRED achieves state-of-the-art accuracy on full-state reconstructions and forecasting of spatiotemporal dynamics from very few sensors.
